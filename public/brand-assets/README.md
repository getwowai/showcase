# WOW AI Brand Assets

This directory contains all the brand assets for WOW AI, organized for easy access and implementation.

## Directory Structure

```
brand-assets/
├── guidelines/           # Brand guidelines and documentation
│   └── BRAND_GUIDELINES.md
├── logos/               # Logo variants and sizes
│   ├── primary/         # Primary logo (Balanced Unity)
│   ├── secondary/       # Secondary logo (Enhanced Big .AI)
│   └── sizes/           # Different logo sizes
├── icons/               # Icon system
│   ├── app/            # App icons (64px)
│   ├── standard/       # Standard icons (32px)
│   └── favicon/        # Favicon (16px)
└── exports/            # Export formats (SVG, PNG, ICO)
    ├── svg/
    ├── png/
    └── ico/
```

## Quick Start

### For Web Development

1. **Primary Logo**: Use `logos/primary/wow-ai-primary-logo.svg` for most applications
2. **Header Logo**: Use `logos/sizes/wow-ai-header-120px.svg` for navigation
3. **Favicon**: Use `icons/favicon/wow-ai-favicon-16px.svg` for browser tabs

### For Mobile Apps

1. **App Icon**: Use `icons/app/wow-ai-app-icon-64px.svg` as base for app store icons
2. **UI Icons**: Use `icons/standard/wow-ai-standard-icon-32px.svg` for interface elements

### For Print Materials

1. **Primary Logo**: Use `logos/primary/wow-ai-primary-logo.svg` (export as PDF for print)
2. **Hero Logo**: Use `logos/sizes/wow-ai-hero-200px.svg` for large displays

## File Naming Convention

All files follow this naming pattern:

- `wow-ai-[variant]-[size].svg`
- Example: `wow-ai-primary-logo.svg`, `wow-ai-hero-200px.svg`

## Color Codes

- **Lime Green (Primary)**: `#aedf1a`
- **Sky Blue (Secondary)**: `#86c9e5`
- **Soft Charcoal (Text)**: `#4a5568`

## Usage Guidelines

### DO:

- ✅ Use Primary Logo for 90% of applications
- ✅ Maintain minimum 20px clear space around logo
- ✅ Use appropriate size variants for context
- ✅ Preserve original proportions and colors

### DON'T:

- ❌ Stretch or distort proportions
- ❌ Change colors without approval
- ❌ Use detailed flower below 32px
- ❌ Add effects like shadows or gradients

## Export Requirements

For production use, you may need these formats:

- **SVG**: For web use (scalable)
- **PNG 24-bit**: For transparent backgrounds
- **ICO**: For favicon implementation
- **PDF**: For print materials

## Support

For questions about brand usage or to request new assets, please refer to the brand guidelines in `guidelines/BRAND_GUIDELINES.md`.
